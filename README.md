Tämä projekti on C#-komentorivisovellus, joka toimii yksinkertaisena kotikirjaston hallintasovelluksena.
Ohjelman avulla käyttäjä voi hallita kirjalistaa konsolin kautta.

Toiminnallisuudet

Lisätä kirjoja kirjalistalle

Kirjalla on nimi, kirjoittaja, julkaisuvuosi ja genre

Poistaa kirjoja listalta

Näyttää kaikki kirjat

Näyttää kirjat genren mukaan

Etsiä kirjoja nimen tai kirjoittajan perusteella

Tekninen toteutus

Toteutettu C#-kielellä komentoriviohjelmana

Kaikki toiminnallisuudet on toteutettu erillisinä metodeina

Pääohjelma ohjaa ohjelman kulkua valikon avulla

Versionhallinta toteutettu Gitillä ja GitHubilla (branchit ja pull requestit)
